Go to terminal and unzip the provided file.

Activate the virtual environment and install the requirements:
source flask/bin/activate
pip install -r requirements.txt

cd into the flask directory. 
cd into the 1_service directory.
Run the command: python3 run.py


Open a new terminal, return to base directory and activate the virtual environment again.
cd into the flask directory.
cd into the web_page directory.
Run the command: python3 run.py

Open web browser and go to: http://127.0.0.1:5000/
